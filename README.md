# Mobile Device Usage Analysis Dashboard

## Project Overview
This project is a beginner-level Power BI dashboard built to analyze mobile device usage patterns of 700 users. The dashboard provides insights into user behavior, screen time trends, and battery consumption patterns.

## Tools & Technologies
- Power BI
- Power Query
- Excel Dataset

## Key Features
- Interactive dashboard with data visualizations
- Analysis of average screen time and app usage
- Battery consumption trends
- User behavior analysis based on age and gender

## Key Insights
- Identified average daily screen time among users
- Observed patterns in app usage across different age groups
- Analyzed battery drain trends based on device usage
- Compared user behavior by gender and age distribution
